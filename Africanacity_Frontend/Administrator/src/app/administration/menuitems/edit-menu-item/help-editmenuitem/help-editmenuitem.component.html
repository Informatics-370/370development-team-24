Help: Editing an Menu Item

If you're unsure how to edit an Menu Item to the system, please follow the steps mentioned below

  • Step 1: Edit the item name should there be any changes that need to made.
  • Step 2: Change the description as desired.
  • Step 3:Select what menu type, the item changed to.
  • Step 4:Select what food type, the item changed to.
  • Step 5:Select what menu category, the item changed to.
  • Step 6: Change the price as desired.
  • Step 7: Confirm the item changes, by clicking the "Confirm" button

    • *Note: Ensure that all fields are filled in correctly, before attempting to save the item

Help: Adding a Drink

If you're unsure how to add an Drink to the system, please follow the steps mentioned below

  • Step 1: Enter in the name of the Drink.
  • Step 2: Input a desired description explaing what the item is.
  • Step 3:Select what drink type, the item is.
  • Step 4:Enter in the "Unit Price"of the item.
  • Step 5: Add the item, by clicking the "Add" button

    • *Note: Ensure that all fields are filled in correctly, before attempting to save the item

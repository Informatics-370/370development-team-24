Help: Adding an Inventory Item

If you're unsure how to add an Inventory Item to the system, please follow the steps mentioned below

  • Step 1: Enter in the name of the Inventory item.
  • Step 2:Select what inventory type, the item is.
  • Step 3:Enter in the "Unit Price"of the item.
  • Step 4: Input a desired description explaing what the item is.
  • Step 5: Enter in the number of items on hand, at the time the item was purchased.
  • Step 6: Add the item, by clicking the "Add" button

    • *Note: Ensure that all fields are filled in correctly, before attempting to save the item

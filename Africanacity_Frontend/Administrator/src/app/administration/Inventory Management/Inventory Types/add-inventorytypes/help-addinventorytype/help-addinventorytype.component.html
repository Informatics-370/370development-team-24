Help: Adding an Inventory Type

If you're unsure how to add a Inventory Type to the system, please follow the steps mentioned below

  • Step 1: Enter in the Name of the Inventory Type.
  • Step 2:Enter in the Description of the Inventory Type.
  • Step 3: Add the item, by clicking the "Add" button

    • *Note: Ensure that all fields are filled in correctly, before attempting to save the item

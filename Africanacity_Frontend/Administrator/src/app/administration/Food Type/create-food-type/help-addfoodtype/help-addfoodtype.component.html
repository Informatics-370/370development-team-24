Help: Adding an Food Type

If you're unsure how to add a Food Type to the system, please follow the steps mentioned below

  • Step 1: Enter in the Name of the Food Type.
  • Step 2:Enter in the Description of the Food Type.
  • Step 3: Add the item, by clicking the "Add" button

    • *Note: Ensure that all fields are filled in correctly, before attempting to save the item

Help: Edit an Entertainment Type

If you're unsure how to edit an entertainment type information to the system, please follow the steps mentioned below

  • Step 1:Edit the name of the Entertainment Type.
  • Step 2:Edit the description of the Entertainment type if necessary.
  • Step 3: Update the entertainment type information, by clicking the "Save" button
  • Step 4: Cancel updating the entertainment type information, by clicking the "Discard" button

    • *Note: Ensure that all fields are filled in correctly, before attempting to save the entertainment type

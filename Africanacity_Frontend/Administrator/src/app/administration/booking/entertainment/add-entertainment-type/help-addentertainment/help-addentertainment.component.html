Help: Adding an Entertainment Type

If you're unsure how to add an Entertainment type to the system, please follow the steps mentioned below

  • Step 1: Enter in the Name of the Entertainment Type.
  • Step 2:Provide a short description of the entertainment type.
  • Step 3: Add the entertainment, by clicking the "Create" button
  • Step 3: Cancel the action, by clicking the "Discard" button

    • *Note: Ensure that all fields are filled in correctly, before attempting to save the entertainment type

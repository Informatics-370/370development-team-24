Help: Edit an Event

If you're unsure how to edit an Event on the system, please follow the steps mentioned below

  • Step 1:Update the title of the Event.
  • Step 2:Change the description.
  • Step 3:Change the event date.
  • Step 4:Change the event cover image.
  • Step 5: Update the event information, by clicking the "Save" button

    • *Note: Ensure that all fields are filled in correctly, before attempting to save the event

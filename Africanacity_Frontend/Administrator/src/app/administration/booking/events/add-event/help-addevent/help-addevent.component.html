Help: Adding an Event

If you're unsure how to add an event to the system, please follow the steps below

  • Step 1: Enter in the Event Title or name
  • Step 2:Enter in a short Description of the event.
  • Step 3:Enter in the date when the event takes place.
  • Step 4:Choose an image you want to upload.
  • Step 5: Add the event, by clicking the "Create" button

    • *Note: Ensure that all fields are filled in correctly, before attempting to save the event

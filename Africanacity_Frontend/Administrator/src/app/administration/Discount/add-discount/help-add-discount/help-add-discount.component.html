Help: Create a Discount Percentage

If you're unsure how to create a discount percentage information to the system, please follow the steps mentioned below

  • Step 1:Enter the name of the Discount percentage.
  • Step 2:Input the description of the discount percentage if necessary.
  • Step 2:Input the amount in decimals of the discount percentage if necessary.
  • Step 3: Create the discount information, by clicking the "Add" button
  • Step 4: Cancel adding the discount information, by clicking the "Cancel" button

    • *Note: Ensure that all fields are filled in correctly, before attempting to save the discount information
